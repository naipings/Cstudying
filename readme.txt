# Cstudying

## 项目简介

这是本人C语言入门学习的代码仓库，包含各种基础语法、数据类型、控制结构、函数、指针、结构体等的实践代码。

## 环境需求

- **C编译器**：建议使用GCC（GNU Compiler Collection）或Clang
- **操作系统**：可在Windows、Linux或macOS上运行
- **编辑器/IDE**：推荐使用VS Code、Vim、Emacs或Code::Blocks

## 使用说明

1. 克隆仓库：

```bash
git clone https://github.com/yourusername/Cstudying.git
```

2. 编译和运行代码：

- 使用GCC编译并运行示例文件：

```bash
# 编译
gcc 001-简单加法.c -o simple_add

# 运行
./simple_add
```

## 学习资源推荐

- 书籍：《C Primer Plus》、《The C Programming Language》
- 在线课程：Coursera C语言入门课程、Udemy C语言教程
- 网站：C语言中文网、菜鸟教程C语言部分

## 问题解决与调试技巧

1. **编译错误**：仔细阅读编译器输出的错误信息，通常会指出错误所在的行和原因。
2. **逻辑错误**：使用`printf`函数输出中间结果，检查程序逻辑是否符合预期。
3. **调试工具**：使用GDB（GNU Debugger）进行调试，设置断点并逐步执行程序。
